# 🏋️‍♂️ VertexFit — Elite Gym Experience Website

**VertexFit** is a premium, fully responsive gym website designed to deliver a high-end digital fitness experience.  
It combines modern UI design, smooth interactions, and performance-focused structure to reflect a professional frontend development approach.

---

🌐 Live Demo: [VertexFit Live](https://maavia971.github.io/vertexfit-gym/)

---

## 🌟 Project Vision

The goal of this project is to simulate a **real-world gym brand website** that:

- Engages users visually  
- Communicates value clearly  
- Encourages action through strong CTAs  
- Maintains clean, scalable, and maintainable code  

---

## ✨ Core Features

- 📱 Responsive Design — Seamless experience across all devices  
- 🎯 Scroll-Based Reveal Animations — Smooth and engaging transitions  
- 🧭 Dynamic Navigation System — Active link highlighting on scroll  
- 🎨 Modern UI Aesthetics — Gradients, glow effects, and glassmorphism  
- 🔝 Sticky Header with Scroll Effects — Enhanced user experience  
- 💳 Interactive Pricing Section — Clearly structured membership plans  
- 🚀 Call-to-Action Integration — Conversion-focused design  

---

## 🧠 Technical Highlights

- Structured using **semantic HTML5**  
- Styled with **advanced CSS3 techniques**
  - Flexbox & Grid layouts  
  - CSS variables for scalability  
  - Custom animations & transitions  
- Built with **Vanilla JavaScript**
  - DOM manipulation  
  - Scroll detection logic  
  - Event-driven UI behavior
